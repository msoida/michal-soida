O stronie

Front-end

Strona jest zgodna ze standardem HTML5 oraz CSS3. Tworząc ją stawiałem przede wszystkim na mały rozmiar i szybkość wyświetlania - dlatego nie ma na niej żadnego kodu javasript, wszystkie graficzne elementy są tworzone za pomocą CSS i SVG.

Aby umożliwić dostęp prostszymi (lub po prostu starszymi) urządzeniami, strona wyświetla się poprawnie w przeglądarkach nie obsługujących arkuszy CSS. Specjalna wersja formatowania jest wykorzystywana do drukowania stron.

Dzięki tym zabiegom cała strona mieści się w 2 KB + 15 KB plików statycznych (style,logo).

Back-end

Server-side software is written in Python, using Flask framework. Source code is available on GitHub.
