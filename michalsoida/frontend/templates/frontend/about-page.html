Front-end

This page is compliant with HTML5 and CSS3 standards. I designed it with small size and display speed in mind - that's why there is no javascript on it, and all graphical elements are created using CSS and SVG.

To allow access with simpler (or simply older) devices, the page displays correctly in browsers that do not support CSS. Also, a special version of formatting is used when printing pages.

Thanks to these measures, the entire site is under 2 KB + 15 KB of static files (styles, logo).

Back-end

Server-side software is written in Python, using Flask framework. Source code is available on GitHub.
